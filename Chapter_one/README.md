# Chapter 1: Python Basics & Environment Setup

This folder covers the absolute fundamentals of setting up Python, configuring environments, and working with basic data types.

## Key Concepts Covered

* **Environment Setup:** Installing Python 3, running scripts via the Linux terminal, and configuring a global `.gitignore`.
* **Variables & Primitive Types:** Working with Strings, Integers, Floats, and Booleans.
* **Basic Input/Output:** Using `print()` and `input()` to interact with the terminal.
* **String Formatting:** Basic manipulation and data type casting (e.g., converting text inputs into integers).
