# Chapter 4: File Handling & External Data

This folder focuses on permanent data persistence—learning how to read, write, and manipulate external files on the system disk using Python.

## Skills Map

* **`text_files.py`** (In Progress) – Mastering context managers (`with open`) and file modes (`r`, `w`, `a`).
* **`json_data.py`** (Pending) – Serializing and deserializing dictionaries into data interchange formats.
* **`csv_analytics.py`** (Pending) – Parsing tabular spreadsheet data using Python's native modules.
