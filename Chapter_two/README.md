# Chapter 2: Control Flow & Data Structures

This directory focuses on making programs smart using conditional logic and organizing data into core collections.

## Key Concepts Covered

* **Conditionals:** Mastering logic branching with `if`, `elif`, `else`, and Python's modern `match-case` statements.
* **Loops:** Using `for` loops to iterate through data and `while` loops to maintain continuous program runtime.
* **Data Collections:** 
  * **Lists:** Ordered, mutable sequences for sequential data.
  * **Dictionaries:** Key-value pairs optimized for fast data lookups.
* **Search Engine Logic:** Utilizing the `.lower()` function and the `in` keyword to build flexible, partial-match search logic.
